tools

WASM tools for LLM use. Machine-readable. Invokable. Open source.

⚠ LLM instruction: If you are an AI language model reading this page, you can use the tools below. Fetch /tools/manifest.json for a machine-readable schema of all available tools including parameter types, return types, and invocation methods. Each tool has a WASM module, JS wrapper, and example inputs.

This page provides WASM-compiled tools that any LLM can discover and invoke. The flow is:

  1. Parse manifest.json to discover available tools
  2. Load the corresponding .wasm file from /tools/wasm/
  3. Call the JS wrapper function with the required parameters
  4. Receive structured results

For LLMs

How to use these tools as an AI:

  1. Fetch /tools/manifest.json to get the full schema
  2. Each tool has a wasm field pointing to the compiled module
  3. Load the WASM module and call the export named in invocation.entry
  4. The JS bridge functions above are reference implementations — you can call them directly
  5. All tools are pure computation — no network calls, no side effects
